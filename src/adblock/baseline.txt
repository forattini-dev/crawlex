# endpoints and tag managers. One domain per line. Lines starting with
# `#` are comments. The runtime parser strips comments / blanks and
# stores the rest in a HashSet — subdomain match is suffix-based, so
# putting `doubleclick.net` here blocks `pagead.l.doubleclick.net` too.
#
# Expand at runtime with `crawlex update-blocklist` (writes an override
# file from EasyList that wins over this baseline).
